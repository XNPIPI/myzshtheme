# myzshtheme
修改者:[纳兰斯坦](http://nalansitan.com/ "纳兰斯坦个人主页")

基于agnoster主题修改的双行zsh主题，添加了如下功能：
* 分不同颜色显示不同目录
* 右侧显示时间和日期
* 原来的单行主题改为双行，更优雅的显示长目录

>因需要计算目录深度，故需要安装bc
